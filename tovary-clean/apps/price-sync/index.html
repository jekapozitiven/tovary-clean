Настройка движка

Скачай файл настроек и положи его в репозиторий (папка sync/). Движок в GitHub Actions читает именно его. Токены сюда не вписываются — они хранятся в GitHub Secrets.

Открыть журнал запусков
ⓘ Как это работает. Робот в GitHub Actions раз в ~10 минут читает товары Black Street и переносит отмеченные поля на Bonna-shop и Core22. Цена, наличие и видимость — почти сразу. Название, артикул, описание и фото — через импорт Prom, поэтому доезжают медленнее. Полная инструкция по запуску — в файле sync/README-sync.md.
